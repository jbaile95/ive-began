# Python Calculator

A beginner-friendly Python project for a simple calculator.

## What it does

- Add, subtract, multiply, divide, and raise to a power
- Handles invalid number input
- Prevents division by zero

## Run it

1. Open a terminal in `python-calculator`
2. Run:

```bash
python main.py
```

## Learn more

- Edit `calculator.py` to add new operations
- Use `main.py` to change how users interact with the calculator
- Use `web.py` and `templates/index.html` to customize the browser UI
